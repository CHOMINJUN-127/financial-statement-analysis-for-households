# financial-statement-analysis-for-households


## 🎯 프로젝트 목적

이 프로젝트는 **가정과학 수업의 발표 활동**을 위해 제작되었다.
가계 재무제표를 사람이 직접 계산하고 정리할 때 발생하는 번거로운 과정을
**파이썬이라는 컴퓨터 프로그래밍 언어를 활용하면 빠르고 효율적으로 처리할 수 있다는 점**을 보여주는 것이 주요 목적이다.

발표 준비 기간이 길지 않았기 때문에,
복잡한 기능보다는 **짧은 코드로도 가계 손익계산서를 빠르게 작성할 수 있는 구조**에 집중하였다.
또한 프로그래밍을 잘 모르는 사람도 코드를 보았을 때
“어렵지 않은데?”, “이 정도면 나도 할 수 있겠다”라고 느낄 수 있도록
**직관적이고 이해하기 쉬운 코드 구성**을 목표로 하였다.

이를 통해 가계 재무 관리와 프로그래밍이 결코 멀리 있는 개념이 아니라,
일상생활에서도 충분히 활용될 수 있다는 점을 전달하고자 하였다.

## 🧾 프로젝트 개요

이 프로젝트는 파이썬을 활용하여 월간 가계 손익계산서를 자동으로 생성하고 분석하는 프로그램이다.
가계의 수입과 지출 항목을 데이터로 정리한 뒤,
총소득, 총지출, 순이익(잔액) 및 저축률을 계산하여
가계 재무 상태를 한눈에 파악할 수 있도록 구성하였다.

또한 계산 결과를 표 형태로 출력하고,
지출 항목별 비율을 원형 그래프로 시각화함으로써
숫자에 익숙하지 않은 사람도 가계 재무 구조를 직관적으로 이해할 수 있도록 하였다.

이 프로젝트는 복잡한 금융 모델이 아닌
일상적인 가계부 수준의 데이터를 기반으로 한 간단한 분석을 목표로 하며,
파이썬을 처음 접하는 사람도 따라 할 수 있는 구조로 설계되었다.
