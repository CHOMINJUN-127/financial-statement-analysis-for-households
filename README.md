# financial-statement-analysis-for-households


## 🎯 프로젝트 목적

이 프로젝트는 **가정과학 수업의 발표 활동**을 위해 제작되었다.
가계 재무제표를 사람이 직접 계산하고 정리할 때 발생하는 번거로운 과정을
**파이썬이라는 컴퓨터 프로그래밍 언어를 활용하면 빠르고 효율적으로 처리할 수 있다는 점**을 보여주는 것이 주요 목적이다.

발표 준비 기간이 길지 않았기 때문에,
복잡한 기능보다는 **짧은 코드로도 가계 손익계산서를 빠르게 작성할 수 있는 구조**에 집중하였다.
또한 프로그래밍을 잘 모르는 사람도 코드를 보았을 때
“어렵지 않은데?”, “이 정도면 나도 할 수 있겠다”라고 느낄 수 있도록
**직관적이고 이해하기 쉬운 코드 구성**을 목표로 하였다.

이를 통해 가계 재무 관리와 프로그래밍이 결코 멀리 있는 개념이 아니라,
일상생활에서도 충분히 활용될 수 있다는 점을 전달하고자 하였다.

## 🧾 프로젝트 개요

이 프로젝트는 파이썬을 활용하여 월간 가계 손익계산서를 자동으로 작성하고 분석하는 프로그램이다.
가계의 수입과 지출 항목을 데이터로 입력하면,
이를 표(DataFrame) 형태로 정리하고 주요 재무 지표를 계산하여
가계 재무 상태를 체계적으로 확인할 수 있도록 구성하였다.

프로그램은 먼저 수입과 지출 데이터를 구분하여 처리한 뒤,
총소득, 총지출, 순이익(잔액)을 계산하고
저축 금액을 기준으로 저축률을 백분율(%) 형태로 산출한다.
이를 통해 단순한 금액 나열이 아니라,
가계 재무 구조를 수치적으로 분석할 수 있다.

또한 계산 결과를 콘솔에 출력하는 데서 그치지 않고,
지출 항목별 비중을 원형 그래프로 시각화하여
숫자에 익숙하지 않은 사람도
어디에 지출이 집중되어 있는지를 직관적으로 이해할 수 있도록 하였다.

이 프로젝트는 복잡한 금융 계산이나 전문적인 회계 지식이 아닌,
일상적인 가계부 수준의 데이터 분석을 중심으로 설계되었으며,
짧은 코드로도 실생활 문제를 해결할 수 있다는 점을 보여주는 데 초점을 두고 있다.

## 🧠 코드 흐름 설명

이 프로젝트의 코드는 **가계 손익계산서 작성 과정을 단계별로 나누어** 구성되어 있다.
각 단계는 실제 사람이 가계부를 작성할 때의 순서를 그대로 따라가도록 설계되어 있어,
프로그래밍 경험이 많지 않은 사람도 전체 흐름을 쉽게 이해할 수 있다.

### ① 데이터 준비

먼저 가계의 수입과 지출 항목을 딕셔너리 형태로 정의한다.
각 항목에는 이름과 금액이 함께 저장되며,
수입은 양수, 지출은 음수로 표현하여 이후 계산이 간단해지도록 하였다.

이 데이터는 pandas의 DataFrame으로 변환되어
가계 손익계산서 표 형태로 정리된다.

### ② 가계 손익계산서 출력

DataFrame으로 변환된 데이터를 콘솔에 출력하여
현재 가계의 수입과 지출 내역을 한눈에 확인할 수 있도록 한다.
이는 실제 손익계산서를 종이에 작성해 보는 과정과 유사한 단계이다.

### ③ 총소득 · 총지출 · 순이익 계산

금액을 기준으로 수입과 지출을 구분하여 합계를 계산한다.

- 금액이 0보다 큰 항목 → 총소득

- 금액이 0보다 작은 항목 → 총지출

총소득과 총지출을 더하여
한 달 동안 남은 금액인 **순이익(잔액)**을 계산한다.
이를 통해 현재 가계 재무 상태를 수치로 확인할 수 있다.

### ④ 지출 데이터 분리 및 전처리

지출 항목만 따로 분리하여 분석에 사용한다.
이때 지출 금액은 시각화를 위해 **절댓값으로 변환**하여
각 항목의 비중이 정확하게 표현되도록 처리하였다.

### ⑤ 저축률 계산

지출 항목 중 ‘저축’을 선택하여
전체 수입 대비 저축이 차지하는 비율을 계산한다.

이를 통해 단순히 돈이 남았는지 여부를 넘어서,
가계 재무 관리가 어느 정도 이루어지고 있는지를
비율(%)로 확인할 수 있다.

### ⑥ 지출 구조 시각화

마지막으로 지출 데이터를 원형 그래프로 시각화한다.
각 지출 항목이 전체 지출에서 차지하는 비율을
한눈에 확인할 수 있도록 하여,
어디에서 지출이 많이 발생하는지를 직관적으로 파악할 수 있다.

## ⚙ 주요 기능

- 가계 수입·지출 항목을 **딕셔너리**로 간단히 입력
- **총수입** / **총지출** / **순수익** 자동 계산
- **저축률(%)** 실시간 산출 (순수익 ÷ 총수입 × 100, 소수점 1자리)
- 지출 항목별 비중을 **파이 차트**로 직관적 시각화 (matplotlib)
- 한국어로 깔끔하게 정렬된 **손익계산서** 콘솔 출력
- 코드 수정 한 번으로 매달/가족별 재사용 가능 (엑셀 없이도 OK)



## 🧮 계산 로직 설명

### 1. 수입 항목 필터링  

   → 값이 `> 0`인 항목만 합산 → `총수입`

### 2. 지출 항목 필터링  

   → 값이 `< 0`인 항목만 합산 → `총지출` (음수 값 유지)

### 3. 순수익 계산  

   python
   순수익 = 총수입 + 총지출   # 지출이 음수이므로 실질적으로 총수입 - |총지출|

### 4. 저축률 계산 

**저축률 = round((순수익 / 총수입) * 100, 1) if 총수입 > 0 else 0**
